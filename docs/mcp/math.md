# Math

Basic arithmetic tools. Both parameters (`a`, `b`) are required floats.

| Tool | Operation |
|------|-----------|
| `add_numbers` | `a + b` |
| `subtract_numbers` | `a - b` |
| `multiply_numbers` | `a × b` |
| `divide_numbers` | `a ÷ b` — returns error if `b` is zero |

## Example prompts

- "What is 42 times 7?"
- "Divide 100 by 3"
